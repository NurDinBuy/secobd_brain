Она возвращает первый индекс элемента массива

По которому индекс элемента совпадает с переданным аргуметом

Но она вернет -1, если такого элемента нет

Пример: 

```
var array = [2, 5, 9];
array.indexOf(2);     // 0
array.indexOf(7);     // -1
array.indexOf(9, 2);  // 2
array.indexOf(2, -1); // -1
array.indexOf(2, -3); // 0
```
