От слова - Dictionary (Словарь)

Создается при помощи фигурных скобок 
Состоит из ключа и значения


Пример:

```
london = {
    'id': 1,
    'name': 'London',
    'it_vlan': 320,
    'user_vlan': 1010,
    'mngmt_vlan': 99,
    'to_name': None,
    'to_id': None,
    'port': 'G1/0/11'
}
```
